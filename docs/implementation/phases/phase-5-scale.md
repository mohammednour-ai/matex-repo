# Phase 5 - Scale (Weeks 40-52)

## Goal
Scale platform footprint: mobile, cross-border, and reliability hardening.

## Deliverables

- React Native mobile app v1:
  - browse listings
  - bidding
  - messaging
  - inspection/photo capture
  - notifications
- Cross-border module:
  - customs docs
  - OFAC screening path
  - CAD/USD conversion handling
- Reliability program:
  - load tests
  - query tuning
  - cache strategy
  - penetration/security audit

## Exit Criteria

- Mobile app published and usable in production.
- At least one successful cross-border transaction.
- Security audit passes with no critical/high findings.

## Release Blockers

- Cross-border flow missing required compliance checks.
- Perf profile failing target under planned concurrency.
- Any unresolved critical security finding.

# archive/

Unreferenced binary assets that were previously stored at the repository root or in
loose folders. Kept here for short-term reference (Phase 1.6, image audit) so they
can be deleted once we are sure nothing else needs them.

Subfolders:

- `repo-root-images/`: stray PNG/JPG files that used to live at the repo root
  (`dashadv.png`, `dashadvs.png`, `MatexLogo.png`, `Phoenix_10_*.jpg`).
  The active assets used by the web app live under `apps/web-v2/public/`
  (`/MatexLogo.png`, `/illustrations/*`, `/hero-features/*`).
- `img-originals/`: original raw screenshots from the previous design pass
  that the hero feature PNGs in `apps/web-v2/public/hero-features/` were
  derived from.

Safe to delete after a release cycle if no one has reached for them.
